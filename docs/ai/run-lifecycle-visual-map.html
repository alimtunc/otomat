Cycle de vie d'une run

Un tour d'agent supervisé. Le parent fork un worker enfant — seul écrivain durable — qui écrit dans events.jsonl ; le parent tail ce ledger et pilote les state machines du domaine. La vérité de fin vient toujours du ledger, jamais d'une supposition.

  • control plane (api · supervisor)
  • worker enfant (durable)
  • ledger / stream
  • retour / async
